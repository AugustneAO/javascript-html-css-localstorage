# This program was made using JavaScript + HTML + css + localStorage

The program receives two numbers and the answer of what will be the result and the operation to be performed (division, multiplication, subtraction and addition). When clicking on the verify button, the program should score the user with 5 values ​​and display the message "Congratulations!" in the Message field if the user's answer matches the expected result, otherwise you must assign -5 values ​​and display the message "Wrong, try again" in the Message field, the score must be displayed in the 'Your Score' field.

When clicking on the restart button, the program should reset the user's score, the first and second numbers, the user's response, the expected result and the message should be empty.
